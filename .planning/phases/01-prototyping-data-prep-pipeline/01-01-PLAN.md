# Plan 01-01: Prototyping OCR Extraction and PDF-to-Image Loading

## Objective

Create a prototype Jupyter Notebook that traverses the Uthai Thani Constituency 2 directory structure, converts PDF pages to images using PyMuPDF, and applies EasyOCR to extract text from a small sample (3-5 PDFs). Validate that OCR can capture Thai characters from the ส.ส.5/18 election forms with acceptable accuracy.

## Context

- Source PDFs are image-based (no embedded text), requiring OCR.
- Target: `source/เขตเลือกตั้งที่ 2/` — nested directories of polling stations.
- Each polling station folder contains `ส.ส.5ทับ18.pdf` and `ส.ส.5ทับ18(บช).pdf`.
- PyMuPDF (fitz) is available for PDF-to-image conversion.
- EasyOCR is available for Thai text recognition.
- Strategy: full-page text extraction with regex keyword matching (Decision D-05, D-06).

## Tasks

### Task 1: Create notebook structure and environment setup cell
**type:** auto

Create a Jupyter Notebook `notebooks/01_prototype_ocr_extraction.ipynb` with:
- Cell 1: Environment imports (os, pathlib, fitz/PyMuPDF, PIL, easyocr, re, json)
- Cell 2: Configuration constants (SOURCE_DIR path, DPI setting, sample size)
- Cell 3: EasyOCR reader initialization (Thai + English languages)

**Done criteria:**
- Notebook file exists at `notebooks/01_prototype_ocr_extraction.ipynb`
- All import cells are syntactically valid Python
- Source directory path points to the correct location

### Task 2: Implement PDF directory traversal
**type:** auto

Add a notebook cell that:
- Recursively walks `source/เขตเลือกตั้งที่ 2/` to find all PDF files matching `ส.ส.5ทับ18.pdf` (election day forms, not advance voting)

- Prints a summary count and first 5 found paths

**Done criteria:**
- Function `find_election_day_pdfs(root_dir)` returns a list of dicts with keys: `path`, `district`, `subdistrict`, `station`
- Summary shows at least 10 PDFs found (the constituency has 250+ stations)

### Task 3: Implement PDF-to-image conversion
**type:** auto

- Uses PyMuPDF (fitz) to render each page at 200 DPI

**Done criteria:**

Add notebook cells that:

**Done criteria:**

### Task 5: Prototype regex extraction on OCR output
**type:** auto

Add a notebook cell that attempts to extract key fields from OCR text:
- Station identifier (หน่วยเลือกตั้งที่ N)
- Total eligible voters (ผู้มีสิทธิเลือกตั้ง)
- Total votes cast (ผู้มาใช้สิทธิ)
- Valid votes count (บัตรดี)
- Candidate vote numbers (numeric rows)

Implement `extract_station_data(ocr_text)` using regex patterns. Run on the 3 sample results.

**Done criteria:**
- Function exists and runs without error on OCR text
- At least one field is successfully extracted from at least one sample
- Results printed as dict per station for manual inspection
